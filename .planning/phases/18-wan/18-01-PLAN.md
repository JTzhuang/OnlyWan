## Type Challenge: Heterogeneous Registry

ModelRegistry must store factories for MULTIPLE unrelated types (CLIPTextModelRunner, T5Runner, WanVAERunner). Two approaches:

**Approach A (recommended):** Type-keyed map inside registry — use `std::type_index` as a secondary key alongside version string. The map stores `std::map<std::string, std::any>` where the value is `std::function<unique_ptr<void>(...)>`. This is complex.

**Approach B (simpler, preferred for this phase):** Template methods with separate per-type maps inside a single singleton. Each call to `register_factory<T>` stores into a `std::map<std::string, FactoryFn<T>>` keyed on type. Implementation uses a `std::map<std::type_index, std::any>` where the `any` holds the per-type `std::map<std::string, FactoryFn<T>>`.

**Approach C (simplest, pragmatic):** Separate per-type registries — `CLIPRegistry`, `T5Registry`, `WanVAERegistry`, each is a singleton `std::map<std::string, FactoryFn<T>>` with its own mutex and `REGISTER_*` macro. No shared registry class needed. Tests call `CLIPRegistry::instance().create(...)` instead of `ModelRegistry::instance()->create<CLIP>(...)`.

Use **Approach C** for this phase: it is the simplest to implement correctly with zero template magic, and gives clear per-type type safety. The REGISTER_MODEL_FACTORY macro becomes per-type: `REGISTER_CLIP_FACTORY`, `REGISTER_T5_FACTORY`, `REGISTER_WAN_VAE_FACTORY`.

For each of the three model types (CLIP, T5, WanVAE), define a registry class following this template:

```cpp
// Template for PerTypeRegistry — inline in header
template<typename ModelType>
class PerTypeRegistry {
public:
    using FactoryFn = std::function<std::unique_ptr<ModelType>(
        ggml_backend_t backend,
        bool offload_params_to_cpu,
        const String2TensorStorage& tensor_map,
        const std::string& prefix
    )>;

    static PerTypeRegistry& instance() {
        static PerTypeRegistry registry;
        return registry;
    }

    void register_factory(const std::string& version, FactoryFn fn) {
        std::lock_guard<std::mutex> lock(mutex_);
        factories_[version] = std::move(fn);
    }

    std::unique_ptr<ModelType> create(const std::string& version,
                                       ggml_backend_t backend,
                                       bool offload,
                                       const String2TensorStorage& map,
                                       const std::string& prefix = "") const {
        std::lock_guard<std::mutex> lock(mutex_);
        auto it = factories_.find(version);
        if (it == factories_.end()) {
            throw std::runtime_error("Unknown model version: " + version);
        }
        return it->second(backend, offload, map, prefix);
    }

    bool has_version(const std::string& version) const {
        std::lock_guard<std::mutex> lock(mutex_);
        return factories_.count(version) > 0;
    }

private:
    PerTypeRegistry() = default;
    mutable std::mutex mutex_;
    std::map<std::string, FactoryFn> factories_;
};
```
