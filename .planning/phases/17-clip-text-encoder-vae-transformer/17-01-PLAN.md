**4. Create tests/cpp/model_factory.hpp** (per D-03 through D-06):

A single-header template factory:
```cpp
template<typename ModelType, typename VersionEnum>
class ModelFactory {
public:
    using CreatorFn = std::function<std::unique_ptr<ModelType>(
        ggml_backend_t backend,
        bool offload_params_to_cpu,
        const String2TensorStorage& tensor_map,
        const std::string& prefix
    )>;

    void register_version(VersionEnum version, CreatorFn creator);
    std::unique_ptr<ModelType> create(VersionEnum version, ggml_backend_t backend,
        bool offload_params_to_cpu, const String2TensorStorage& tensor_map,
        const std::string& prefix = "") const;
    bool has_version(VersionEnum version) const;
    std::vector<VersionEnum> registered_versions() const;
private:
    std::map<VersionEnum, CreatorFn> registry_;
};
```

- `create()` throws `std::runtime_error("Unknown model version")` if version not registered (per D-06)
- Include `#pragma once`, needed headers: `<functional>`, `<map>`, `<memory>`, `<vector>`, `<string>`, `<stdexcept>`
- Include `"ggml_extend.hpp"` and `"ggml-backend.h"` for `ggml_backend_t` and `String2TensorStorage`
- All methods inline (header-only template)
