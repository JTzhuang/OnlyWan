<review_notes>
## ODR Safety (confirmed safe)

Including model headers (clip.hpp, t5.hpp, vae.hpp, flux.hpp) directly from test
files is SAFE for this codebase. All member functions are defined within struct/class
bodies (implicitly inline in C++). The linker deduplicates identical inline definitions
from test TUs and wan-api.cpp. This is standard C++ behavior, not an ODR violation.

## Empty String2TensorStorage Verified

Model constructors call `model.init(params_ctx, storage_map, prefix)` which calls
`init_params()` + `init_blocks()`. These create tensors in params_ctx regardless of
whether storage_map has entries. The map is only used for type inference via `get_type()`,
which falls back to `GGML_TYPE_F16` when a tensor name is not found.
Therefore `alloc_params_buffer()` succeeds with empty storage for ALL 4 model types.

## Enum Values Verified (from src/model.h)

- VERSION_FLEX_2 (line 42) -- CORRECT, not a typo for FLUX_2
- VERSION_FLUX2 (line 49) -- separate enum value, used for VAE
- VERSION_FLUX, VERSION_FLUX_FILL, VERSION_CHROMA_RADIANCE, VERSION_OVIS_IMAGE -- all verified
- CLIPVersion enum: OPENAI_CLIP_VIT_L_14, OPEN_CLIP_VIT_H_14, OPEN_CLIP_VIT_BIGG_14 -- verified

## Destruction Order

ALWAYS declare BackendRAII BEFORE any Runner or unique_ptr<Runner> in the same scope.
C++ destroys local variables in reverse declaration order. Runner destructor frees internal
buffers (params_buffer, compute_buffer, params_ctx) but does NOT free runtime_backend.
BackendRAII destructor calls ggml_backend_free() on the backend handle.
Correct order: Runner destroyed -> BackendRAII destroyed -> backend freed.

1. Create `BackendRAII guard(ggml_backend_cpu_init())` -- FIRST (before any Runner)

3. Get factory from `register_clip_factory()`
4. Call `auto runner = factory.create(version, guard.backend, false, empty_map, "")`
5. Call `runner->alloc_params_buffer()` -- works with empty storage (model.init() creates tensors regardless)
6. `WAN_ASSERT_TRUE(runner != nullptr)`
7. `WAN_ASSERT_EQ(runner->get_desc(), std::string("clip"))`

**Test function** -- `test_clip_factory_roundtrip(TestSuite& suite)`:
- "all_3_versions_registered": Register all 3, verify `registered_versions().size() == 3`
- "create_each_version": Create each version, verify all return non-null with correct desc
