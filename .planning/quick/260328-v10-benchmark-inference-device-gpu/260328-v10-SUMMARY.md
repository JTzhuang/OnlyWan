# Quick Task 260328-v10: GPU Device Selection for benchmark_inference

## Summary

Added `--device` parameter support to the benchmark_inference tool, enabling users to select specific GPU device IDs when using CUDA backend without code changes.

## Changes Made

### Modified Files
- `tests/cpp/benchmark_inference.cpp`

### Implementation Details

1. **Updated `backend_from_string()` function signature**
   - Added `int device_id = 0` parameter
   - Modified CUDA initialization to use `ggml_backend_cuda_init(device_id)` instead of hardcoded device 0
   - Updated debug output to show selected device ID

2. **Updated all benchmark functions**
   - `benchmark_clip()`: Added `int device_id = 0` parameter
   - `benchmark_t5()`: Added `int device_id = 0` parameter
   - `benchmark_vae()`: Added `int device_id = 0` parameter
   - `benchmark_transformer()`: Added `int device_id = 0` parameter
   - All functions now pass `device_id` to `backend_from_string()`

3. **Updated main() function**
   - Added `int device_id = 0;` variable initialization
   - Added argument parsing for `--device` parameter: `else if (arg == "--device" && i + 1 < argc) { device_id = std::stoi(argv[++i]); }`
   - Updated all benchmark function calls to pass `device_id` parameter
   - Added help text: `"  --device ID                             GPU device ID for CUDA backend (default: 0)\n"`

## Verification

All changes verified:
- `device_id` variable properly initialized and parsed from command line
- `backend_from_string()` signature updated with device_id parameter
- All four benchmark functions (CLIP, T5, VAE, Transformer) pass device_id to backend selection
- Help text documents the new `--device` option
- Default behavior (no --device flag) uses device 0

## Usage Examples

```bash
# Use default device (0)
./benchmark_inference --backend cuda --model clip

# Use specific GPU device
./benchmark_inference --backend cuda --device 1 --model t5

# Multi-GPU benchmarking
./benchmark_inference --backend cuda --device 0 --output results_gpu0.csv
./benchmark_inference --backend cuda --device 1 --output results_gpu1.csv
```
